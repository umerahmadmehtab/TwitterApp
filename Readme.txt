Description: A simple twitter app using angular JS and php. As the page is loaded, email and tweet of a person is required. The user enters both 
and as a result a tweet is submitted to database. Previous tweets are also fetched from the database.
Use localhost to run the code, also import the database in phpmyadmin or mysql workbench for proper functionality of the app. Do not forget to change the 
username and password according to your settings.
